# Warcraft III Ability 关联数据

该目录由 `export-object-editor-data.mjs` 从经典版原始 SLK 与 Func/Strings 生成。

- ability_metadata：AbilityMetaData 字段定义、本地化名称和逐 Ability 绑定。
- buff_effect_editor_data：AbilityBuffData 以及 Ability 引用的 profile-only Buff/Effect 闭包。
- ability_editor_data/coverage-manifest.json：单位引用、行为家族、召唤、科技、Buff/Effect 与元数据覆盖基线。
